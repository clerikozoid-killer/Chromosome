# `golden/` — эталонные снапшоты (регрессия)

Снапшоты ожидаемых результатов (`syrupy`). Каждый подтверждённый провал
превращается в постоянный golden-тест — это и есть «обучение» детерминированной
системы (см. `docs/TESTING_AND_MENTOR.md` §0, §3.2).

Наполняется по мере реализации слоёв. Сейчас — пусто.
